# S3

## Create instance

```bash
docker run -d --name fustfs \
  -p 9000:9000 \
  -p 9001:9001 \
  -e RUSTFS_ACCESS_KEY=username \
  -e RUSTFS_SECRET_KEY=secret123 \
  -e RUSTFS_REGION=us-east-1 \
  -e RUSTFS_DATA_DIR=/data \
  -v fustfs-data:/data \
  rustfs/rustfs:latest
```

create bucket

```bash
AWS_ACCESS_KEY_ID=username AWS_SECRET_ACCESS_KEY=secret123 aws --endpoint-url http://localhost:9000 s3 mb s3://bucket1
```

```bash
docker run -d --name minio \
  -p 9000:9000 \
  -p 9001:9001 \
  -e MINIO_ROOT_USER="username" \
  -e MINIO_ROOT_PASSWORD="secret123" \
  -v minio-data:/data \
  quay.io/minio/minio server --console-address ":9001" /data
```

goto [http://127.0.0.1:9001](http://127.0.0.1:9001)

login with `username` and `secret123`

create `bucket1`

## Use

```bash
aws s3 ls s3://bucket1/ --endpoint-url http://localhost:9000
```

## aws cli

#### install

```bash
python3 -m pip install awscli
```

### set env variables

```bash
cat > .env <<_EOF_
AWS_CA_BUNDLE=/path/to/root/ca_root.pem
AWS_ACCESS_KEY_ID=<id>
AWS_SECRET_ACCESS_KEY=<key>
_EOF_

export $(cat .env | xargs)
```

### configure

```bash
aws configure set aws_access_key_id <id>
aws configure set aws_secret_access_key <key>

cat ~/.aws/credentials
```

### list files

```bash
aws s3 ls s3://bucket-name/ \
     --endpoint-url=https://s3.url-address.ru \
     --ca-bundle /path/to/root/ca_root.pem \
     --no-verify-ssl
```

### copy files

```bash
aws s3 cp s3://bucket-name/ ~/local/ \
   --recursive \
     --endpoint-url=https://s3.url-address.ru \
     --ca-bundle /path/to/root/ca_root.pem \
     --no-verify-ssl
```
